# Clothing Website Backend

This project is a backend scaffold for a clothing e-commerce application.

## Scripts

- npm start
- npm run dev

## Structure

- src/app.js: application bootstrap
- src/server.js: server entry point
- src/config: database and environment configuration
- src/features: feature-based modules for auth, users, products, categories, cart, orders, wishlist, reviews, coupons, uploads, and dashboard
- src/middlewares: authentication, validation, error handling, uploads, and admin guards
- src/utils: shared helpers and utilities
